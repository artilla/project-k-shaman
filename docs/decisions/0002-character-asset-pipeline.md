# ADR-0002 — 캐릭터 에셋 파이프라인: 베타는 AI 정지컷 + CSS 연출, 리깅은 v1.1 지표 조건부

- 상태: 제안 (운영자 스타일 선정 시 확정)
- 날짜: 2026-07-07
- 관련: `fortune-engine/character-sheet-hongyeon.md` §6 (비주얼 정본), `docs/ux/screen-ia.md` S4 (아바타 상태 FSM), master-spec §1.5 (베타 = 홍연 1종)

## 맥락

S4 무대 화면(T022)은 아바타 상태 4종(greeting/idle/speaking/blessing)을 요구하지만 에셋이 없어
플레이스홀더(수정구슬)로 동작 중이다. 베타 기한은 2026-07-20. 캐릭터 시트 §6에 비주얼 방향
(붉은 단청, K-pop 무대 + 전통 미감 + 퇴마 판타지, lucky.color 팔레트 6색)은 확정돼 있다.

## 선택지

| 안 | 내용 | 기한 내 | 비용 | 리스크 |
|---|---|---|---|---|
| ① AI 정지컷 + CSS/JS 연출 | 상태별 일러스트 4컷 + 크로스페이드·bob·음량 글로우 | O | ~수$ | 컷 간 캐릭터 일관성 |
| ② Live2D/스켈레톤 리깅 | 립싱크·연속 모션 | X | 외주/수주 | 베타 범위 초과 |
| ③ AI 비디오 루프 | 상태별 짧은 루프 영상 | △ | 중 | 일관성·용량·루프 이음새 |
| ④ 일러스트 외주 | 사람 작화 | X | 중~고 | 일정 |

## 결정 (제안)

**베타는 ①.** 정지컷 4 + 공유카드용 1컷. speaking의 "음량 기반 입 모양"(S4 명세)은
WebAudio AnalyserNode 음량 → 아바타 글로우/스케일 펄스로 **근사**한다 (립싱크 아님을 명시).
②는 베타 지표(청취 완료율 60%)가 모션 투자를 정당화할 때 v1.1에서 재평가.

## 결과

- 일관성 리스크 완화: 단일 "캐릭터 지문" 프롬프트(의상·머리·소품 고정) + 같은 세션/시드에서
  상태 변주만 — T023 프롬프트 팩으로 정본화.
- 라이선스: 상업 이용 가능한 생성 도구만 사용, 도구·생성일·프롬프트를 에셋 매니페스트에 기록.
- 안전: 실존 인물/아이돌 유사성 금지, 공포 연출 금지 (시트 §6 "무섭지 않게").
- 성능 예산: 투명 WebP 1024px, 장당 ≤150KB, lazy preload — 3초 재생 경로 무영향 (§1.6-②).
- 폴백 불변식: 에셋 부재 시 T022 플레이스홀더로 동작 — 에셋은 진행을 막지 않는다.
