핵심 UX 결정
텍스트 먼저, autoplay 금지 — S4에서 운세 텍스트는 API 응답 즉시 노출, 음성은 "듣기" 탭으로만 시작 (v3 §11 준수).
글래스는 절제해서 — 반투명 블러는 플레이어와 공유 바텀 시트 등 임시 레이어에만. 본 콘텐츠는 불투명 카드.
공유는 바텀 시트 — S6를 별도 화면 대신 S5 위의 바텀 시트로 처리해 흐름 이탈 최소화.
아바타 상태 크로스페이드 — idle → greeting → speaking → blessing 4종 에셋을 300ms 페이드로 전환, 재생 세그먼트와 동기화.
게스트 우선 온보딩 — 가입 없이 1탭 시작, 소셜 로그인은 보조. 개인정보는 로컬 우선 + 해시 전송 안내.
