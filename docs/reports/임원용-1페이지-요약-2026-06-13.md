# 오늘신당(K-Shaman) 임원용 1페이지 요약

작성일: 2026-06-13
근거: `docs/planning/today-shindang-service-plan-v3.md` §3/§8/§13/§14/§19, `docs/reports/경쟁분석-보고서-2026-06-13.md`

## 결론

**현재 MVP 범위로 진행 권장.** 2026-06-13 기준 경쟁 재검증 결과, "매일 1분짜리 캐릭터 음성 운세 공연" 포지션은 비어 있다. 점신·포스텔러는 운세 정보 플랫폼, 헬로우봇은 캐릭터 운세 챗봇에 가깝고, 오늘신당의 핵심 차별점인 **TTS 음성 + 캐릭터 모션 + 공유 가능한 결과물**을 동시에 제공하는 서비스는 확인되지 않았다.

## 시장 판단

| 항목 | 판단 | 시사점 |
| --- | --- | --- |
| 포지셔닝 | 시청각 엔터테인먼트 운세 사분면 공백 | MVP 범위 변경 불필요 |
| 최근접 경쟁자 | 헬로우봇: 캐릭터 IP·구독·유저 기반 보유 | TTS/음성 운세 출시 여부를 최우선 감시 |
| 기리고 | 넷플릭스 시리즈 홍보용 소품 앱, 상시 경쟁자 아님 | 영상형 의례 UX 수요 신호 + 대형 IP 진입 위협 사례 |
| 점신·포스텔러 | 콘텐츠 폭·리텐션 강점, 캐릭터 음성 퍼포먼스 약함 | 정보량 경쟁은 피하고 공연성에 집중 |

## MVP 유지 범위

**집중할 것:** 홍연 1종, 오늘의 운세 생성, 텍스트 먼저 노출, 사전합성 우선 TTS 재생, 기본 모션/음량 기반 립싱크, 결과 카드, 공유 이미지, 기본 이벤트 분석.

**넣지 않을 것:** 실시간 상담 챗봇, 실제 무속인 연결, 캐릭터 2종 이상, 정밀 음소 립싱크, 3D/VRM, 무제한 장문 리포트, 영상 공유. 경쟁사 콘텐츠 폭을 따라가기보다 "3초 첫 반응 + 1분 공연 + 공유 카드"의 품질을 우선한다.

## 수익성 기준

| 기준치 | 현재 가정 |
| --- | --- |
| 헬로우봇 정액권 | 26,000원/월 |
| 오늘신당 캐릭터 패스 | 2,900~3,900원/월 |
| 무료 운세 cache miss 원가 | 약 $0.0090/건 (약 12.4원) |
| 10k DAU 정식 캐시 60% 월 변동비 | 약 $1,175/월 (약 162만 원) |
| 변동비 break-even | MAU 유료 전환 약 1.54% |

헬로우봇의 26,000원/월 정액권은 캐릭터 기반 운세의 지불 의향 근거다. 오늘신당의 2,900~3,900원 캐릭터 패스는 보수적 가격 가설이며, 5단계 수익화 실험에서 가격 수용성과 전환율을 함께 재계산한다. (원화 기준 환율: $1 = ₩1,380 가정)

## 주요 리스크와 대응

| 리스크 | 대응 |
| --- | --- |
| 헬로우봇이 기존 캐릭터에 TTS/음성 운세를 추가 | 분기 1회 + 신년 시즌 전 재점검, 베타 일정 준수, 홍연 IP·연출 품질 선점 |
| 캐시 히트율 미달로 원가 상승 | 베타 캐시 30% 이상을 게이트로 관리, provider usage 로그로 실청구액 교체 |
| 대형 IP의 운세×캐릭터 앱 진입 | 상표·아트워크·보이스 계약 조기 확보, 오리지널 세계관 강화 |
| 운세 정보량 비교 열위 | 정보 플랫폼 경쟁을 피하고 "1분 공연" 프레임 유지 |

## 임원 의사결정 요청

1. **MVP 범위 유지 승인:** 경쟁사 대응 명목의 기능 추가 없이 홍연 1종·음성 재생 UX·공유 카드에 집중.
2. **0단계 검증 승인:** Gemini 공식 `count_tokens`, provider usage 로그, 고정 운영비, 보이스 계약 범위를 실제 수치로 교체.
3. **모니터링 체계 지정:** 헬로우봇 음성/TTS 출시, 대형 IP 운세 앱 발표, 점신·포스텔러 캐릭터 IP 도입을 분기 1회와 신년 시즌 전 점검.
